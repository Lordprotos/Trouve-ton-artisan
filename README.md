# Trouve-ton-artisan

Application Angular pour trouver des artisans.

## Développement

Pour démarrer le serveur de développement :

```bash
ng serve
```

L'application sera accessible sur `http://localhost:4200/`.

## Scripts disponibles

- `ng serve` - Démarre le serveur de développement
- `ng build` - Compile l'application pour la production
- `ng test` - Lance les tests unitaires
- `ng lint` - Analyse le code

## Structure du projet

```
src/
├── app/
│   ├── app.ts              # Composant principal
│   ├── app.html            # Template principal
│   ├── app.scss            # Styles principaux
│   ├── app.routes.ts       # Configuration des routes
│   └── app.config.ts       # Configuration de l'application
├── styles.scss             # Styles globaux
└── index.html              # Point d'entrée HTML
```
